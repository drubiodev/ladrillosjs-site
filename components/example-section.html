See It In Action

counter.html 
<div>
  <h2>Count: {count}</h2>
  <button onclick="increment()">
    Add
  </button>
  <button onclick="reset()">
    Reset
  </button>
</div>

<script>
  let count = 0;

  const increment = () => {
    count++; // Triggers re-render
  };

  const reset = () => {
    count = 0;
  };
</script>

Automatic Re-rendering

State changes automatically trigger component updates. No need to call setState or manage subscriptions manually.

  • ✓ Direct variable assignment
  • ✓ Automatic change detection
  • ✓ Minimal re-renders
  • ✓ No boilerplate code
form.html 
<div>
  <h2>Hello, {name}!</h2>
  <input 
    type="text"
    $bind="name"
    placeholder="Your name"
  />

  <p>Email: {email}</p>
  <input $bind="email" />

  <label>
    <input 
      type="checkbox" 
      $bind="subscribe"
    />
    Subscribe: {subscribe}
  </label>
</div>

<script>
  let name = "World";
  let email = "";
  let subscribe = false;
</script>

Two-Way Data Binding

Use the $bind attribute for automatic synchronization between inputs and state. Works with text inputs, checkboxes, selects, and textareas.

  • ✓ No manual event handlers
  • ✓ Instant synchronization
  • ✓ Works with all form elements
  • ✓ Clean, declarative syntax
login-form.html 
<button onclick="login()">
  Login
</button>

<script>
  const login = () => {
    // Emit event to other components
    $emit("user-logged-in", {
      userId: 123,
      username: "john"
    });
  };
</script>
header.html 
<header>
  <span $if="{isLoggedIn}">
    Welcome, {username}!
  </span>
</header>

<script>
  let isLoggedIn = false;
  let username = "";

  // Listen for events
  $listen("user-logged-in", (user) => {
    isLoggedIn = true;
    username = user.username;
  });
</script>

Global Event Bus

Communicate between components without prop drilling. Emit events from one component and listen in others.

  • ✓ Cross-component communication
  • ✓ Decoupled architecture
  • ✓ Type-safe event data
  • ✓ Automatic cleanup
cart.html 
<div>
  <h1>Shopping Cart</h1>

  <p $if="{items.length === 0}">
    Your cart is empty
  </p>

  <div $else-if="{items.length < 5}">
    <p>You have {items.length} items</p>
  </div>

  <div $else>
    <p>Cart is full! ({items.length})</p>
  </div>

  <ul>
    <li $for="item in items">
      {item.name}
    </li>
  </ul>
</div>

<script>
  let items = [];
</script>
