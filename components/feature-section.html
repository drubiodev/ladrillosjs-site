Built on web standards with modern features that just work.

Templating

Simple syntax. Reactive by default.

Bind any JavaScript expression with {curly braces}. When state changes, the DOM updates automatically — no virtual DOM, no diffing ceremony, no build step.

  • Plain {expression} interpolation — no directives to memorize
  • Top-level variables are reactive state
  • Standard inline handlers like onclick — zero framework syntax
Two-Way Binding

Form state, instantly in sync.

Connect any input to a variable with $bind. Edits flow both ways — update the input and your state changes, update your state and the input follows. No event handlers, no boilerplate.

  • Works with text, email, checkbox, and more
  • Automatic type coercion — booleans stay booleans
  • One attribute replaces value + oninput plumbing
References

Direct DOM access, when you need it.

Tag any element with $ref and reach it through $refs. Perfect for focusing inputs, measuring layout, integrating third-party libraries, or anything else that needs the real DOM node.

  • One declarative attribute — no querySelector calls
  • Access the real DOM node for focus, scroll, measurements
  • Drop in any third-party library that needs an element
Control Flow

Render what matters, when it matters.

Branch your UI with <if>, <else-if>, and <else> — or toggle visibility without unmounting using <show>. Declarative, readable, and exactly what you'd expect from plain HTML.

  • <if> / <else-if> / <else> mount and unmount branches
  • <show> keeps the element in the DOM — ideal for preserving state
  • Any JavaScript expression works — ternaries, method calls, the works
List Rendering

Loops that just make sense.

Render any array with $for. Mutate the list and the DOM keeps up — add items, remove them, reorder them — it all works without lifecycle hooks, keys, or re-renders you have to think about.

  • Familiar (item, i) in items syntax — destructuring supported
  • Reassign the array and the list updates — no manual diffing
  • Pairs seamlessly with $if for empty states and filters
Cross-Component Comms

Skip the prop drilling. Just send events.

Broadcast events from anywhere with $emit, and pick them up with $listen. Siblings, cousins, deeply-nested components — they all talk without touching a shared parent.

  • $emit(name, payload) broadcasts to any listener
  • $listen(name, handler) subscribes from anywhere in the tree
  • No stores, no context providers, no callback chains
